# Спортивный трекер

Веб-приложение для отслеживания физической активности и спортивных тренировок. Позволяет регистрировать тренировки, отслеживать прогресс и получать статистику в виде графиков.

## Функциональные возможности

- Добавление тренировок (бег, плавание, силовые)
- Ввод параметров: длительность, дистанция, интенсивность, дата
- Автоматический расчёт сожжённых калорий
- Общая статистика (количество тренировок, общая дистанция, калории)
- График прогресса за последние 7 дней (Chart.js)
- Сохранение данных в базе данных (SQLite)

## Технологический стек

| Технология | Назначение |
|------------|------------|
| Python + Flask | Серверная логика и API |
| SQLite + SQLAlchemy | Хранение данных |
| HTML + CSS | Интерфейс |
| JavaScript + Chart.js | Графики и динамика |

## Запуск проекта

1. Клонировать репозиторий

git clone https://github.com/sberezowskaya/sport-tracker.git
cd sport-tracker

2. Создать виртуальное окружение

python3 -m venv venv
source venv/bin/activate (для Mac/Linux)
venv\Scripts\activate (для Windows)

3. Установить зависимости

pip install flask flask-sqlalchemy

4. Запустить приложение

python3 app.py
